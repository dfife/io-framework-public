# Paper 31 Public Bundle

The initial Paper 31 bundle in this repository is focused on the practical
fixed-IO `C_l` calculator and its immediate diagnostics.

Included public artifacts:

- practical IO `C_l` confrontation script and report
- practical baryon-slot audit script and report
- result JSON / CSV tables
- residual figure

The current public bundle is intentionally narrow. It is designed to expose the
core practical calculator state without mirroring the full private research tree.
