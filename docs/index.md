# IO Framework Public Resources

This repository is the public evidence layer for the Interior Observer (IO)
framework.

It is intended to hold:

- reproducible scripts
- frozen reports
- data products
- figures
- provenance manifests

Current public focus:

- Paper 31 practical calculator and audit bundles
- calculator Phase 3 perturbation-foundation release slice
